# Kartlands
***A Mario Kart mod for Gaslands***

This set of rules will allow you to play games of Mario Kart on the tabletop. It is not a complete set of rules, however,
but rather a modification to [Gaslands](https://planetsmashergames.com/gaslands/about/). As such, you will need a copy
of the game in order to play this mod.

These rules are in active development, and changes are to be expected. I am using LEGO Mario Kart sets for this, and have
scaled the movement templates to use for 28mm Gaslands, as described in the rules book. Feel free to scale according to
whatever vehicles you have available.
